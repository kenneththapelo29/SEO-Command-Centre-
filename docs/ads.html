Google Ads

Pull search terms and keyword data from Google Ads.

What gets fetched

FileContents
data/ads/search_terms.jsonSearch terms with impressions, clicks, cost, conversions
data/ads/keywords.jsonKeyword bids with impression share

Authentication

Google Ads requires separate OAuth setup. Run the setup script after getting a developer token.

python scripts/setup_auth.py

Developer tokens require an application at Google Ads API Center. Approval takes 24-48 hours.

No API access yet?

Download search terms as CSV from the Google Ads UI: Reports → Predefined reports → Search terms. Drop the CSV in data/ads/ and the fetcher will convert it automatically.

Fetch

python run_fetch.py --sources ads
Store google-ads.yaml in the project root. It is in .gitignore and must never be committed.
