Generating Reports

Turn your data into client-ready reports.

Markdown reports

Ask Claude Code to write a report and save it to the reports/ folder:

Write a full monthly SEO report from all data sources.
Save it to reports/report-2026-03.md

The report will be saved as a markdown file you can open in any editor or convert to PDF.

CSV output

Ask Claude Code to generate a CSV for any analysis:

Turn the paid-organic gap analysis into a CSV
I can send to the client.

Google Docs output

Claude Code can generate properly formatted markdown that you paste into Google Docs. For automatic conversion, consider adding a google-docs-forge integration.

Verify before sending

Always spot-check Claude Code outputs against the source JSON files before sending to a client. LLMs can occasionally misread data. If a number looks surprising, check the raw file.

Trust but verify. Check surprising numbers against source JSON before anything goes to a client.
