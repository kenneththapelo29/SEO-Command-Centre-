Configuration

Edit config.json to connect your site and APIs.

config.json reference

{
  "name": "Your Site Name",
  "domain": "yourdomain.com",
  "gsc_property": "https://www.yourdomain.com/",
  "ga4_property_id": "319491912",
  "google_ads_customer_id": "",
  "industry": "Your Industry",
  "competitors": ["https://competitor1.com/"],
  "ai_visibility_source": "bing_csv",
  "ai_visibility_keywords": ["your brand", "main keyword"],
  "google_forms": [
    {"title": "Contact Form Responses", "output": "contact_leads.json"}
  ]
}

Field reference

FieldDescriptionExample
nameYour site or client nameAcme Corp
domainBare domain, no httpsacme.com
gsc_propertyExact GSC property URL including trailing slashhttps://www.acme.com/
ga4_property_idNumeric GA4 property ID from Admin settings319491912
google_ads_customer_idAds customer ID, numbers only1234567890
ai_visibility_sourcedataforseo, serpapi, searchapi, direct_llm or bing_csvbing_csv
gsc_property must exactly match your Search Console property URL including whether it has www and a trailing slash.
