SEO Command Center

Claude Code-powered SEO analysis. Connect your Google APIs. Ask questions. Get answers.

What is this?

SEO Command Center is a Python toolkit that pulls live data from Google Search Console, GA4, Google Ads, Google Forms and AI visibility platforms into local JSON files — then lets Claude Code read and cross-reference everything at once.

Instead of exporting CSVs from three different platforms, VLOOKUPing them together in Excel and spending an afternoon on analysis, you run one command and ask questions in plain English.

"Which keywords am I paying for that I already rank for organically?" — answered in 90 seconds instead of an afternoon.

How it works

1

Connect your APIs

One-time setup (~1 hour). Create a Google Cloud service account and grant it read access to your GSC and GA4 properties.

2

Fetch your data

Run python run_fetch.py --sources gsc,ga4,ads,forms,ai. Data lands in JSON files in your project directory.

3

Ask Claude Code anything

Open Claude Code in the project directory. It reads CLAUDE.md automatically which contains your full prompt library. Ask questions — each one builds on the last.

Quick start

git clone https://github.com/kenneththapelo29/SEO-Command-Centre-.git
cd SEO-Command-Centre-
pip install -r requirements.txt

Then follow the Installation guide to connect your APIs.

New to this? Start with GSC only — it is the easiest API to connect and gives you immediate value. Add GA4, Ads and Forms once that is working.

Project structure

├── run_fetch.py         # Master fetch runner
├── requirements.txt     # Python dependencies

│   ├── fetch_ga4.py

│   ├── fetch_forms.py
